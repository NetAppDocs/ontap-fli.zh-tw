= 執行 ONTAP FLI Solaris 遷移後主機補救
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
遷移後、Solaris 主機需要採取補救措施，以確保新導入的 LUN 能夠被正確識別並正常運作。



== 遷移後 - 離線 FLI

對於離線 FLI，刪除 LUN 匯入並確認主機啟動器已連線後，將 LUN 上線。

| 步驟 2：識別多重路徑裝置 | 步驟 2：識別多重路徑裝置 | 步驟 2：識別多重路徑裝置 | 步驟 2：識別多重路徑裝置 | 步驟 2：在 SPARC 伺服器 OBP 上，設定提供給此主機 HBA 的新開機裝置

NOTE: 請參閱作業系統廠商文件以取得最新和更新的命令清單。

.步驟
. 附加後重新掃描。
+
[source, cli]
----
cfgadm -al
----
. 識別儲存裝置。
+
[source, cli]
----
format
sanlun lun show all
mpathadm list lu
mpathadm show lu
----
. 驗證裝置是否已連接。
+
[source, cli]
----
format |grep disk_id
----
. 匯入 zpool 後，FS 將自動掛載。
+
[source, cli]
----
zpool import zpool_name
df -g (to ensure its unmounted)
----
