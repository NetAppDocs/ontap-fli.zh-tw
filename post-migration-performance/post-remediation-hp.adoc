== 遷移後 - 線上 FLI

刪除 LUN 匯入後，檢查 LUN 是否已設定 `-space-reserve disabled`和 `-space-allocation enabled`。如果沒有，請相應地進行修改。

為確保 SCSI UNMAP 功能正常運作，請查閱供應商文件以確認支援的軟體版本。啟用此功能需要 Veritas VxVM + VxFS 組合。

|===
| 檔案系統 | 區塊裝置 | LVM （FS） | Sanboot

.步驟
. 識別儲存裝置。
+
[source, cli]
----
ioscan -fnC disk
insf -e
sanlun lun show all
sanlun lun show all -pv
sanlun lun show -wwpn
----
. 匯入 VG 並將 VG 上線。
+
[source, cli]
----
vgimport /dev/vg_name /dev/disk/disk_name
vgchange -a y /dev/vg_name
----
. 識別磁碟區群組和檔案系統。
+
[source, cli]
----
vgdisplay -v /dev/vg_name
vgdisplay /dev/vg_name
lvdisplay /dev/vg_name/lv_name
----
. 掛載檔案系統。
+
[source, cli]
----
bdf
mount /dev/vg_name/vol_name /mount_point
----
