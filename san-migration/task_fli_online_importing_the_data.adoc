= 使用ONTAP FLI 在線遷移從外部陣列導入數據
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
建立 LUN 匯入關係並將主機連線從外部陣列移至ONTAP陣列後，您可以將資料從外部來源 LUN 匯入到ONTAP目標 LUN。

.步驟
. 將權限等級設定為進階。
+
[source, cli]
----
set -privilege advanced
----
. 當系統詢問您是否要繼續時、請輸入「y」。
. 開始匯入移轉。
+
[source, cli]
----
lun import start -vserver <SVM_name> -path <volume_path|storage_unit_path>
----
. 顯示FLI狀態。
+
[source, cli]
----
lun import show -vserver <SVM_name> -path <volume_path|storage_unit_path>
----
