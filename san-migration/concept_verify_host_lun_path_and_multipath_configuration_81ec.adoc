= 驗證主機LUN路徑和多重路徑組態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在進行任何移轉之前、請確認多重路徑的設定正確、而且運作正常。

LUN的所有可用路徑都應為作用中。請參閱SAN主機多重路徑驗證主題、以取得如何驗證Windows、Linux和ESXi主機上多重路徑的範例。
